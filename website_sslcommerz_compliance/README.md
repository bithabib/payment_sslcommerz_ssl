# Website SSLCommerz Compliance

Odoo 19 module for SSLCommerz payment gateway compliance requirements in Bangladesh.

## Features

- **Policy Pages**: Terms & Conditions, Return & Refund Policy, Privacy Policy, About Us, Delivery Information
- **Unified Footer**: Single footer with Policies, Company info, Business Info, Delivery, Social Media, and Payment Banner
- **Payment Banner**: Official SSLCommerz "We Accept" banner showing all payment methods
- **Stock Display**: Show stock quantity on product pages (optional)
- **Terms Agreement**: Require customers to agree to terms at checkout (optional)
- **Configurable Settings**: All options configurable from Website Settings

## Installation

1. Copy the module to your Odoo addons path
2. Update the apps list
3. Install "Website SSLCommerz Compliance" from Apps

## Configuration

Go to **Website → Configuration → Settings → SSLCommerz Compliance** to configure:

- Company Name
- Trade License Number
- TIN Number
- Registered Address
- Contact Email & Phone
- Delivery Times
- Display options (Trade License, Stock Quantity, Terms Agreement, Payment Banner)
